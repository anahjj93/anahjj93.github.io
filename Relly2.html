WRC 世界汽车拉力锦标赛2

谁参加WRC2?

WRC2是所有拉力赛2规格赛车的主场，这是这项运动中 仅次于 混合动力拉力赛1赛车的第二梯队。需要注意的主要车辆有:

  • 雪铁龙 C3
  • 福特 嘉年华
  • 现代 i20
  • 斯柯达 晶锐
  • 大众 polo GTI

规则

WRC赛季的每一轮都包含WRC2级别，但车手和副车手只能提名七个得分轮。他们最好的六个成绩算。

WRC2挑战者*

WRC2挑战者(以前称为WRC2 Junior)现在向WRC2注册的车手和副车手开放，这些车手之前没有在WRC级别的拉力赛中获得过冠军。之前获得WRC制造商积分提名的车手也不符合资格。这与去年不同，当时的资格是根据年龄而不是经验确定的。

WRC大师杯*

规则的改变意味着WRC大师杯现在向驾驶Rally2、Rally3、Rally4、Rally5和RGT赛车的车手和副车手开放。参赛选手年龄必须在50岁及以上。

WRC2车队锦标赛是如何运作的？

一些参加WRC2的车手也将为车队得分，比如Toksport和PH Sport。每个车队可以提名两名车手得分。车队最多可以参加欧洲的六场拉力赛和一场长途赛事。他们在欧洲赛场上的五次最佳成绩和长途跋涉将决定他们的最终排名。

是否有Power Stage点?

是的，在周末的最后阶段，三位最快的车手将分别获得3分、2分和1分，以提高他们的总积分榜。

在WRC2的参赛表上，车手名字旁边的字母代表什么?

    在参赛名单上，车手配对旁边的字母通知了他们被提名的类别。以下是每个字母的含义:
  • D = 车手冠军
  • C = 副车手冠军
  • T = 团队冠军
  • CD = 挑战者冠军车手
  • CC = 挑战者锦标赛的副车手

WRC3在2024年如何运作?

WRC3锦标赛是专门为认可的拉力赛赛车举办的。目前只有M-Sport波兰的福特嘉年华拉力赛3是合格的，然而，雷诺的Clio拉力赛3预计将在4月底获得认可。车手和副车手可以提名五项比赛得分，计算他们最好的四项成绩。

国际汽联青年WRC怎么样?

国际汽联少年WRC分五轮举行。它适用于1994年1月1日或之后出生的车手，使用由M-Sport波兰准备的相同的福特嘉年华拉力赛3汽车。青少年WRC仍然是一个独立的世界锦标赛，但参赛者也有资格获得WRC3积分，如果他们愿意的话。
